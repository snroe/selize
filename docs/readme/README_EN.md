# Snet

[English](docs/readme/README_EN.md)

## Installation

```sh
pnpm add snet -g
```

## Usage
```sh
snet create

✔ Input project name:  test
✔ Select packageManager:  pnpm
```
Then wait a moment, snet will download the template from the GitHub repository and generate the project.

## Issues
